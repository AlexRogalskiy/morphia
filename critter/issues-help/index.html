Bugs / Feature Requests

Please provide as much information as possible (including version numbers) about the issue type and how to reproduce it. Ideally, if you can create a reproducer for the issue at hand, that would be even more helpful. To help with this, please take a look at the reproducer project. Pull Requests

We are happy to accept contributions to help improve Critter. We will guide user contributions to ensure they meet the standards of the codebase. Please ensure that any pull requests include documentation, tests, and also pass the build checks.

To get started check out the source and work on a branch:

$ git clone https://github.com/MorphiaOrg/morphia.git
$ cd morphia
$ git checkout -b myNewFeature

Finally, ensure that the code passes all the checks.

$ cd morphia
$ mvn -Pquality
